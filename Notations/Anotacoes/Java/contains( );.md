## O metodo contains( ) esta de alguma forma relacionado com o metodo equals ( )

==Sim==, o método `contains()`está diretamente relacionado ao método [equals()](https://www.google.com/search?q=equals%28%29&oq=metodo+contains+esta+relacionado+com+o+metodo+equals%28+%29+%3F+&gs_lcrp=EgZjaHJvbWUyBggAEEUYOTIHCAEQIRigATIHCAIQIRigATIHCAMQIRigATIHCAQQIRigATIHCAUQIRigAdIBCTM4NDA0ajBqNKgCALACAQ&sourceid=chrome&ie=UTF-8&mstk=AUtExfAvBP4oOLs04YvX14wrEUrqCWvTJ7kd3TIJxnOGUt9yAN50iS2GoumMZQLhppVZEKcT3J7TspXCLVhVpOmhaUIYBRzJLexR4dyoCSqXoQCajeOgogYcSAsIEyw9IVdyh6555Z2U1EqllAI0UlTfgg1lLw5ha7d4b-b03P5UNHMONJE&csui=3&ved=2ahUKEwiau42boOaSAxVudUEAHdlwHFQQgK4QegQIARAC) em Java. Quando `contains()`é usado em coleções ( `List`, `Set`), ele examina os elementos e chama o método `equals()`de cada objeto para determinar se o item procurado é idêntico a algum elemento existente.
